ICollections is a set of Generics Interfaced Collections for Delphi.

The idea is to build interface collections from scratch to Delphi, focusing on performance, usability and memory allocation.

Planned Support:

- IList
- ILinkedList
- IDictionary
- IStack
- IQueue
